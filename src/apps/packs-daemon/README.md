# Blackmount "Packs" package manager

## Package Config
Name of the package
``` conf
name=...
```

Type of the package
0 = Executable
1 = Service
``` conf
type=0/1
```

The version of the package in string format
``` conf
strv=1.0.0
```

The version of the package in int format
``` conf
strv=100
```
